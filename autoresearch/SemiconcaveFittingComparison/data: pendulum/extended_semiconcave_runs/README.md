# Pendulum Swing-Up: Extended Semiconcave Runs

This folder stores the later pendulum swing-up run set with semiconcave-labeled
outputs. It is part of the semiconcave fitting comparison, not a separate
top-level experiment.

- `pmp/results.tsv` compares PMP runs.
- `transient/results.tsv` compares transient runs.
- `results.tsv` is the combined local scoreboard.

Generated per-run JSON files and logs are local artifacts and are ignored by
Git.
