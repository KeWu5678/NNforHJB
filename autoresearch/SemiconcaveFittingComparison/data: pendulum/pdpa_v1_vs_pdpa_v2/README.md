# Pendulum Swing-Up: PDPA_v1 vs PDPA_v2

This folder stores the direct pendulum swing-up comparison between `PDPA_v1`
and `PDPA_v2` on PMP and transient datasets.

- `pmp/results.tsv` compares PMP runs.
- `transient/results.tsv` compares transient runs.
- `results.tsv` is the combined local scoreboard.

Generated per-run JSON files and logs are local artifacts and are ignored by
Git.
