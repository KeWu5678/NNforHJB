# Pendulum Swing-Up

Pendulum swing-up results for the semiconcave fitting comparison.

- `pdpa_v1_vs_pdpa_v2/` contains the direct `PDPA_v1` versus `PDPA_v2`
  comparison on PMP and transient datasets.
- `extended_semiconcave_runs/` contains the later semiconcave-labeled run set.
  It is kept separate because its result files overlap with the direct
  comparison and include duplicate `PDPA_v2` baseline rows.

Both slices use `pmp/` and `transient/` subdirectories.
